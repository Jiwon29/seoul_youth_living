# 🏠 청년 1인가구, 서울 어디에서 살면 좋을까?
### 서울 1인가구 주거환경 대시보드

> 안전 · 편의 · 가격 3가지 지표로 분석한 서울 25개 자치구 주거환경 비교 대시보드


---

##  프로젝트 개요

이번 학기를 시작하며, 첫 자취를 시작하게 되었습니다. 자취 환경을 고려할 때, 다양한 질문들이 있었습니다. 내 집의 위치가 안전한지, 주변에 마트나 편의점이 얼마나 가까이에 얼마나 많이 있는지, 월세 가격은 어떻게 형성되어있는지 등 고려할 점들이 수없이 많았기에 어느 구에서 살면 좋을지 간단하게 질문에 답변해줄 대시보드의 필요성을 느꼈습니다. 학생인 지금은 학교 위치와 다양한 것들을 고려해야해서 자치구를 정하는 것은 어렵지 않았지만 앞으로 졸업하고 새로 살 곳을 정할 때에는 더욱더 다양한 선택지가 기다리고 있을 것이라 생각했고, 지속적으로 오르는 월세 비용과 생활 인프라를 비교해보면 좋을 것 같다는 생각에 해당 대시보드를 작성하게 되었습니다. 

서울에서 자취를 시작하는 청년 1인가구가 "어느 구에서 살면 좋을까?"라는 질문에 데이터로 답하는 대시보드입니다.  
단순한 월세 비교를 넘어, **안전 인프라 · 생활 편의시설 · 주거 비용** 세 가지 축을 종합적으로 분석해 각각 데이터를 바탕으로 자치구별 주거환경 점수를 산출했습니다.

##  데이터 출처

| 테이블 | 설명 | 출처 |
|--------|------|------|
| `safety` | 서울시 지구대·파출소 위치 (233개) | 경찰청 공공데이터포털 |
| `convenience` | 서울시 편의점·슈퍼마켓·대형마트 위치 (17,376개) | 소상공인 상가업소 데이터 (공공데이터포털) |
| `monthly_rent` | 서울시 자치구별 월세 실거래가 (661,645건) | 서울시 전월세 실거래가 |
| `score` | 자치구별 인프라 점수 (집계·산출) | 위 3개 테이블 가공 |

## 데이터 클렌징 과정
**1. monthly_rent 테이블 (서울특별시 전월세가)**

원본데이터 )
파일 형식: CSV
원본 컬럼 23개 (접수년도, 자치구코드, 자치구명, 법정동코드, 법정동명, 지번구분코드, 지번구분, 본번, 부번, 층, 계약일, 전월세구분, 임대면적, 보증금, 임대료, 건물명, 건축년도, 건물용도, 계약기간, 신규계약구분, 갱신청구권사용, 종전보증금, 종전임대료)

클렌징 내용
- 처리 항목내용컬럼 축소23개 → 3개 (자치구명, 전월세구분, 임대료_만원)행 
- 필터링전월세구분 == "월세" 인 행만 유지
결과: 661,645행 / 3개 컬럼 → monthly_rent 테이블 저장

**2. safety 테이블 (지구대·파출소)**

원본 데이터 )
원본 컬럼 6개 (연번, 시도청, 경찰서, 관서명, 구분, 주소)

클렌징 내용
- 항목내용지역 필터링주소에 "서울" 포함된 행만 유지자치구명 추출서울 25개 구 리스트와 매칭해 주소에서 자치구명 파싱컬럼 추가 
- 자치구명 파생 컬럼 생성컬럼 추가 : 25개 구 리스트와 매칭해 주소에서 자치구명 파싱컬럼 추가
- 카카오맵 REST API 호출로 주소 → 위도 / 경도 변환 지오코딩 실행.
결과: 233행 / 6개 컬럼 → safety 테이블 저장

**3. convenience 테이블 (편의점·마트)**

원본데이터 )
원본 컬럼 38개 (소상공인진흥공단 상가업소 전국 데이터)

클렌징 내용
- 항목내용컬럼 축소38개 → 5개 (상호명, 상권업종소분류명, 시군구명, 위도, 경도)업종 
- 필터링상권업종소분류명 기준 편의점·슈퍼마켓·대형마트만 유지지역 필터링시군구명이 서울 25개 구인 행만 유지파생 컬럼 생성시설유형 컬럼 생성 (편의점 / 마트로 통일)
- 컬럼명 변경상호명 → 시설명, 시군구명 → 자치구명결측치 제거위도/경도 NULL인 행 drop (지도 시각화 불가 행 제거)
결과: 17,376행 (편의점 9,443 / 슈퍼마켓 7,933) / 5개 컬럼 → convenience 테이블 저장

**4. score 테이블 (인프라 점수 - 파생 테이블)**

1. 자치구별 집계
   safety      → 자치구별 안전시설 개수 COUNT
   convenience → 자치구별 편의시설 개수 COUNT
   monthly_rent → 자치구별 평균 월세 AVG (임대료 > 0 필터)

2. 3개 테이블 INNER JOIN (자치구명을 기준으로)

3. MIN-MAX 스케일링 (1~10점 척도)

4. 가중 합산 → 총점
   총점 = 안전점수×3 + 생활점수×3 + 가격점수×4 (최대 100점)
결과: 25행 (서울 25개 구) / 8개 컬럼 → score 테이블 저장
---

##  인프라 점수 산출 방법

각 지표를 **Min-Max 스케일링**으로 0~10점으로 정규화한 뒤 합산했습니다.

```
안전점수  = MinMax(자치구별 안전시설 수)         × 3   ↑ 높을수록 좋음
생활점수  = MinMax(자치구별 편의시설 수)         × 3   ↑ 높을수록 좋음
가격점수  = (1 - MinMax(자치구별 평균 월세))     × 4   ↓ 낮을수록 좋음 → 반전 처리

총점 = 안전점수 + 생활점수 + 가격점수 (최대 100점)
```

>  월세는 낮을수록 유리하기 때문에 스케일링 후 `1 - scaled` 로 반전 처리했습니다.

##  대시보드 구성

| 섹션 | 설명 |
|------|------|
| **사이드바** | 자치구 선택 드롭다운 + 선택 구의 요약 지표 카드 |
| **지도** | 자치구별 평균 월세 색상 표시, 호버 시 상세 정보 / 선택 구의 시설 위치 포인트 표시 |
| **시설 막대차트** | 선택 구의 시설 유형별 (지구대·파출소·편의점·마트) 개수 |
| **레이더 차트** | 선택 구의 안전·생활·가격 점수를 서울 평균과 비교 |
| **파이 차트** | 선택 구의 안전·생활·가격 점수를 서울 평균과 비교 |
| **TOP 5 막대차트** | 인프라 총점 기준 상위, 하위 5개 자치구 및 항목별 구성 |
| **히트맵** | 25개 구 × 3개 점수 전체 강약점 패턴 한눈에 비교 |

##  분석 결과 요약

| 순위 | 자치구 | 총점 | 특징 |
|------|--------|------|------|
| 1 | 강서구 | 72.1점 | 월세 저렴 + 편의시설 풍부 |
| 2 | 영등포구 | 67.2점 | 편의시설 많고 교통 편리 |
| 3 | 종로구 | 66.2점 | 안전시설 압도적 1위 |
| 4 | 송파구 | 64.5점 | 편의시설 최상위권 |
| 5 | 구로구 | 60.9점 | 월세 저렴 + 편의시설 양호 |



### 핵심 인사이트

- 강서구는 안전·생활·가격 세 항목이 모두 평균 이상으로, 특정 항목에 치우치지 않은 1위 자치구입니다.
- 강남구는 편의시설 개수 서울 1위이지만, 평균 월세 131만원으로 가격점수가 1.71점에 불과해 총점은 10위에 그칩니다.
- 종로구는 지구대·파출소 수가 서울 최다지만 편의시설이 적어 생활점수가 2점대로 생활 시설이 약점입니다.
- 용산구는 평균 월세 138만원으로 서울 1위를 기록, 가격점수 1.0점으로 인프라 총점 꼴찌입니다. 상대적으로 월세 가격이 높은 한남동 등이 영향을 많이 미쳤을 것이라 생각됩니다.
- 서초구도 월세가 용산구 다음으로 높아 가격점수 1.21점에 불과, 총점 하위권에 위치합니다.
- 구로구,중랑구,금천구는 가격점수 최상위권으로, 청년 1인가구의 주거비 부담이 가장 낮은 실속형 자치구입니다.
- 전반적으로 가격점수(가중치 ×4)가 총점에 가장 큰 영향을 미쳐, 강남권(강남·서초·용산)은 높은 생활 인프라에도 불구하고 하위권에 위치합니다.
