# 🏠 청년 1인가구, 서울 어디에서 살면 좋을까?
### 서울 1인가구 주거환경 대시보드

> 안전 · 편의 · 가격 3가지 지표로 분석한 서울 25개 자치구 주거환경 비교 대시보드


---

## 📌 프로젝트 개요

서울에서 자취를 시작하는 청년 1인가구가 "어느 구에서 살면 좋을까?"라는 질문에 데이터로 답하는 대시보드입니다.  
단순한 월세 비교를 넘어, **안전 인프라 · 생활 편의시설 · 주거 비용** 세 가지 축을 종합적으로 분석해 자치구별 주거환경 점수를 산출했습니다.

---

## 📊 데이터 출처

| 테이블 | 설명 | 출처 |
|--------|------|------|
| `safety` | 서울시 지구대·파출소 위치 (233개) | 경찰청 공공데이터포털 |
| `convenience` | 서울시 편의점·슈퍼마켓·대형마트 위치 (17,376개) | 소상공인 상가업소 데이터 (공공데이터포털) |
| `monthly_rent` | 서울시 자치구별 월세 실거래가 (661,645건) | 서울시 전월세 실거래가 |
| `score` | 자치구별 인프라 점수 (집계·산출) | 위 3개 테이블 가공 |

---

## 🧮 인프라 점수 산출 방법

각 지표를 **Min-Max 스케일링**으로 0~10점으로 정규화한 뒤 합산했습니다.

```
안전점수  = MinMax(자치구별 안전시설 수)         × 3   ↑ 높을수록 좋음
생활점수  = MinMax(자치구별 편의시설 수)         × 3   ↑ 높을수록 좋음
가격점수  = (1 - MinMax(자치구별 평균 월세))     × 4   ↓ 낮을수록 좋음 → 반전 처리

총점 = 안전점수 + 생활점수 + 가격점수 (최대 100점)
```

> 💡 월세는 낮을수록 유리하기 때문에 스케일링 후 `1 - scaled` 로 반전 처리했습니다.

## 🗺️ 대시보드 구성

| 섹션 | 설명 |
|------|------|
| **사이드바** | 자치구 선택 드롭다운 + 선택 구의 요약 지표 카드 |
| **지도** | 자치구별 평균 월세 색상 표시, 호버 시 상세 정보 / 선택 구의 시설 위치 포인트 표시 |
| **시설 막대차트** | 선택 구의 시설 유형별 (지구대·파출소·편의점·마트) 개수 |
| **레이더 차트** | 선택 구의 안전·생활·가격 점수를 서울 평균과 비교 |
| **파이 차트** | 선택 구의 안전·생활·가격 점수를 서울 평균과 비교 |
| **TOP 5 막대차트** | 인프라 총점 기준 상위, 하위 5개 자치구 및 항목별 구성 |
| **히트맵** | 25개 구 × 3개 점수 전체 강약점 패턴 한눈에 비교 |

## 🏆 분석 결과 요약

| 순위 | 자치구 | 총점 | 특징 |
|------|--------|------|------|
| 1 | 강서구 | 72.1점 | 월세 저렴 + 편의시설 풍부 |
| 2 | 영등포구 | 67.2점 | 편의시설 많고 교통 편리 |
| 3 | 종로구 | 66.2점 | 안전시설 압도적 1위 |
| 4 | 송파구 | 64.5점 | 편의시설 최상위권 |
| 5 | 구로구 | 60.9점 | 월세 저렴 + 편의시설 양호 |

> ⚠️ 본 점수는 3가지 지표만을 반영한 참고용 분석입니다. 교통, 문화시설, 학군 등 다양한 요소를 추가로 고려해야 합니다.

### 핵심 인사이트
서울 25개 구 주거환경 
- 강서구는 안전·생활·가격 세 항목이 모두 평균 이상으로, 특정 항목에 치우치지 않은 균형형 1위 자치구입니다.
- 강남구는 편의시설 개수 서울 1위(생활점수 10.0점)이지만, 평균 월세 131만원으로 가격점수가 1.71점에 불과해 총점은 10위에 그칩니다.
- 종로구는 지구대·파출소 수가 서울 최다(안전점수 10.0점)이지만 편의시설이 적어 생활점수(2.52점)가 뚜렷한 약점입니다.
- 용산구는 평균 월세 138만원으로 서울 1위를 기록, 가격점수 1.0점으로 인프라 총점 꼴찌(13.60점)입니다.
- 서초구도 월세(136만원)가 용산구 다음으로 높아 가격점수 1.21점에 불과, 총점 하위권(22위)에 위치합니다.
- 구로구·중랑구·금천구는 가격점수 최상위권(9~10점)으로, 청년 1인가구의 주거비 부담이 가장 낮은 실속형 자치구입니다.
- 전반적으로 가격점수(가중치 ×4)가 총점에 가장 큰 영향을 미쳐, 강남권(강남·서초·용산)은 높은 생활 인프라에도 불구하고 하위권에 위치합니다.
